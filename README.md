# RT
> Ray Tracer

Implementing the Ray-Tracing method to create a computer generated image.

![](yo2.png)
![](yo1.png)
![](yo3.png)

Bonuses:

• Multi-spots

• Shine effects

• Making screenshot

• Regulating transparency and reflection

• Composed elements

• External files for scene description

• Limited objects

• Ambiance light

• Direct light

• More native elements

• Filters

## Installation

OS X & Linux:

```sh
make && make clean
```

## Usage example

```sh
./RT scenes/[any file]
```
